# ⚡ Automation Scripts

Collection of useful automation scripts for web scraping, data processing, and task automation.

## 📂 Scripts

### 1. Web Scraper
`web_scraper.py` - Extract data from websites

```bash
python web_scraper.py
```

### 2. File Organizer
`file_organizer.py` - Automatically organize files by type

```bash
python file_organizer.py
```

### 3. Image Downloader
`image_downloader.py` - Download images from URLs

```bash
python image_downloader.py
```

## 🛠️ Requirements

```bash
pip install requests beautifulsoup4 selenium
```

## 📝 Features

- Web scraping with BeautifulSoup
- File automation
- Image downloading
- Data processing
