# 🔓 JavaScript Deobfuscator

A tool to decode and analyze obfuscated JavaScript code.

## Features

- Decode Base64 encoded strings
- Unwrap eval() calls
- Analyze LZString compressed data
- String array reconstruction
- Clean code output with comments

## Installation

```bash
npm install
```

## Usage

```bash
node deobfuscator.js <input-file.js>
```

## Example

```bash
node deobfuscator.js sample.js
```

## Output

Generates `output_clean.js` with readable code.
